# Budget Tracker

## Overview
The Budget Tracker is a simple web-based application that helps users efficiently manage their financial records by tracking expenses and monitoring their budget. It provides an intuitive interface to log, manage, and visualize spending habits.

## How It Works
1. **Home Page (`Home.html`)**
   - Acts as the main landing page where users get an overview of the Budget Tracker.
   - Users can navigate to the expense tracking page for detailed logging and analysis.
   
2. **Expense Tracking Page (`Tracker.html`)**
   - Allows users to enter their expenses manually.
   - Displays the list of recorded transactions.
   - Provides a simple way to monitor and categorize spending.

3. **Styling and UI Enhancements (`Home.css`, `Tracker.css`)**
   - The application has a clean and responsive UI with custom styling.
   - Icons and images enhance the user experience.

## Features
- **User-Friendly Interface**: Easy-to-use web pages with a clean design.
- **Expense Logging**: Users can manually input and track their expenses.
- **Categorization**: Expenses can be categorized for better financial planning.
- **Visual Indicators**: Icons and images help in easy identification of different expense types.
- **Basic Financial Overview**: Allows users to see a quick summary of their spending habits.


## Tech Stack Used
- **HTML**: Provides the structure and layout of the web pages.
- **CSS**: Enhances the styling, layout, and responsiveness of the application.

## How to Use
1. Open `Home.html` in a browser to access the main page.
2. Click on the link to navigate to the expense tracking page (`Tracker.html`).
3. Enter your expense details and categorize them accordingly.
4. View a summary of your recorded transactions to analyze spending habits.

## Future Enhancements

- **Persistent Storage**: Implement a database or local storage for saving records.
